# Stängelispass 🍺

**Version**: 2.0.0 | **Status**: Production Ready 🚀

A social beer tracking app with real-time features, gamification, and delightful UX.

## 🚀 Quick Start

```bash
# Install dependencies
cd app && npm ci

# Run tests
npm test

# Start development server
npm run start

# Type check
npm run typecheck

# Lint
npm run lint
```
