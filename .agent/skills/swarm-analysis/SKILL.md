# Swarm Analysis

This skill encapsulates the swarm intelligence logic originally handled by the Swarm Agents.

## Capabilities
- **Roadmap Analysis**: Analyzes the project roadmap and suggests updates.
- **Documentation Sync**: Synchronizes documentation across the codebase.
- **Feature Brainstorming**: Generates new feature ideas based on current trends and codebase.
- **Weekly Analytics**: Performs a weekly deep dive into project progress.

## Usage
Use the provided workflows to trigger swarm analysis:
- `run-swarm`: Runs the main swarm analysis script.
- `update-roadmap`: Triggers a roadmap-specific analysis.
- `sync-docs`: Synchronizes documentation using swarm agents.
