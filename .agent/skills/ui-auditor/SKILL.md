# UI/UX & Aesthetics Auditor

This skill ensures that all UI changes meet the project's high standards for aesthetics ("Premium" feel) and accessibility.

## Capabilities
- **Design Token Audit**: Verifies usage of centralized colors, typography, and spacing.
- **Micro-animation Review**: Checks for smooth transitions and hover effects.
- **Accessibility Check**: Ensures proper use of semantic HTML and ARIA labels.
- **Visual Consistency**: Identifies deviations from the project's established design language.

## Design Principles
- **Vibrant & Harmonious**: No plain red/blue; use the curated project palette.
- **Micro-animations**: Every interaction should feel responsive and alive.
- **Modern Typography**: Use the project's custom fonts (Inter/Outfit) exclusively.

## Usage
Use the provided workflows:
- `audit-ui`: Performs a design and accessibility audit on specific components.
