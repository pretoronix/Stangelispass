# Maintainability Refactor Swarm

This swarm is focused on refactoring and maintainability improvements without behavior changes.

## Agents
1. Refactor Agent (🏗️)
2. Maintainability Auditor (🧭)
3. Dependency Curator (📦)
4. Regression Guard (🧪)

## Workflow
1. Audit: Detect hotspots, smells, dependency risks
2. Proposal: Define refactor candidates and success metrics
3. Safety: Identify tests required for refactor safety
4. Consensus: Weighted vote and prioritization
5. Execution: Apply refactors and document changes

## Outputs
- Prioritized refactor plan
- Risk and test coverage assessment
- Applied refactor changes (if approved)

## How to Run
```bash
npm run swarm:analyze maintainability_refactor
```
